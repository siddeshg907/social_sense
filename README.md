# Social Sense Frames
A Pages UI with help of Next.js App.Created two Frames, Dashboard and Theme Selector.

## Deployed Link
**Frontend** https://social-sense-two.vercel.app/

## Tech Stack:
- Next.js

## Setup Instructions
1. Clone the repository from GitHub.
2. Navigate to the project directory in the terminal.
3. Install dependencies:
```bash
npm install
```
4. Start the frontend development server:
```bash
npm run dev
```
5. Access the application in your web browser.


## Dashboard Frame

### Features Implemented
1. *Charts*
   - Rendered the all charts data dynamically.

2. *Consumer Table*
   - The consumer table has search functionality by name and amount.

## Theme Selector

### Features Implemented
1. *Sidebar*
   - Shown responsive sidebar 

2. *Theme Colours*
   - The Theme colours can be selected and able to change the theme of profile page
   - It has Solid,Linear Gradients,Radial Gradients colours for theme

3. *Profile*
  - Their is profile section with data.
  - The Selected colour will change the theme of profile.
